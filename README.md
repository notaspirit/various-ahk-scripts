# various-ahk-scripts

This is a collection of various ahk scripts that I personally use and that have broader appeal.

## Scripts
- **FormatGithubLinks**: Formats GitHub links in the active clipboard to `[PR|ISSUE NUMBER: PR|ISSUE TITLE](<LINK>)`, `[USER/REPONAME: FILENAME](<LINK>)`, `[USER/REPONAME](<LINK>)`, `[USER/REPONAME: MILESTONENAME Milestones](<LINK>)` or `[USER](<LINK>)`.
    - Default Hotkey: `Shift + Win + Q`
- **FormatWindowsFilePaths**: Formats Windows file paths in the active clipboard by resolving env vars.
  - Example: `C:\Users\user\AppData\Roaming\REDModding\WolvenKit` -> `%appdata%\REDModding\WolvenKit`
  - Default Hotkey: `Shift + Win + A`
